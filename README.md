# Sanatsal - Infinite Visual Synthesizer

Sanatsal is a high-performance Android visualizer capable of generating infinite 3D worlds and fluid simulations in real-time.

## Features

*   **Infinite 3D Worlds:** Uses Raymarching to generate procedurally infinite landscapes (Geometric, Neon Rings, Abstract).
*   **Fluid Simulation:** Interactive 2D fluid dynamics with bloom and vorticity support.
*   **AMOLED / Void Mode:** Ultra-optimized dark mode for OLED screens, saving battery and looking cool.
*   **Customization:** Full control over colors, density, speed, and rendering quality via a glass-morphism Settings Panel.
*   **Audio Reactivity:** (Experimental) Visuals react to microphone input.

## Tech Stack

*   **Language:** Kotlin
*   **Graphics:** OpenGL ES 3.0 (Custom GLSL Shaders)
*   **UI:** Programmatic Android Views (No XML Layouts)

## Setup

1.  Clone the repository.
2.  Open in Android Studio.
3.  Build and Run on an Android device (supports Android 10+).
